h1. auth_media Plugin for REDAXO Community AddOn 2.8.x

Dieses Repository als ZIP Archiv herunterladen, entpacken und den Ordner in "auth_media" umbennenen. Anschließden in /redax/include/addons/community/plugins/ hochladen und über das REDAXO Backend (AddOns) installieren.

Download a ZIP archive of this repository, extrude it and rename the folder to "auth_media". Then upload it to /redaxo/include/addons/community/plugins. Install it on the REDAXO backend site addons.
